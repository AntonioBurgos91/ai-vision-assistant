¿Dónde obtengo mi API key?

Ayuda

¿Cómo usar la aplicación?

  1. Configura tu API key de Anthropic arriba
  2. Vuelve a la página principal
  3. Da instrucciones en lenguaje natural al asistente
  4. Revisa el plan de acciones y confirma la ejecución

Ejemplos de instrucciones:

  • "Abre el bloc de notas y escribe Hola Mundo"
  • "Busca Python en Google"
  • "Copia el texto seleccionado y pégalo en un nuevo archivo"
  • "Toma una captura de pantalla y guárdala"

Características:

  • Visión por computadora: La IA puede ver tu pantalla y entender el contexto
  • Automatización: Controla mouse, teclado y aplicaciones
  • Seguridad: Revisas todas las acciones antes de ejecutarlas
  • Flexible: Usa lenguaje natural para dar instrucciones
Seguridad

La aplicación tiene acceso para controlar tu computadora. Revisa siempre las acciones antes de confirmar su ejecución.
